Política de Privacidade — VidaExtra®

Última atualização: 15/11/2025

O VidaExtra é uma aplicação web que auxilia no cálculo de horas extras (AC-4) e na organização de eventos através do Google Calendar. Esta política explica quais dados coletamos, como usamos, armazenamos e protegemos suas informações.

Dados que coletamos

  • Dados de conta Google via Firebase Auth: nome, e‑mail e foto.
  • Tokens do Google Calendar: access token e refresh token para sincronização de eventos.
  • Eventos criados pelo usuário (título, descrição, local, data e hora) quando você optar por sincronizar com o Google Calendar.

Como usamos seus dados

  • Autenticar seu acesso ao aplicativo.
  • Ler e criar eventos no Google Calendar somente mediante seu consentimento explícito.
  • Gerar lembretes e histórico de cálculos.

Bases legais

O tratamento é realizado com base no seu consentimento durante o login e autorização do Google Calendar.

Armazenamento e segurança

  • Tokens são armazenados com segurança no Firebase/Firestore e usados unicamente para a finalidade de sincronização de agenda.
  • Empregamos controles de acesso e comunicação segura (HTTPS).

Compartilhamento

Não vendemos nem compartilhamos seus dados com terceiros. O acesso é restrito aos serviços do Google para autenticação e calendário.

Seus direitos

  • Revogar o acesso ao Google Calendar a qualquer momento.
  • Solicitar exclusão dos seus dados armazenados pelo VidaExtra.

Conformidade com políticas do Google

O VidaExtra cumpre a Google API Services User Data Policy. Os dados do Google Calendar são utilizados apenas para fornecer funcionalidades do aplicativo e não são usados para publicidade.
